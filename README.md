# 🧠 MyET AI News Navigator

🚀 An AI-powered, multilingual, and interactive news intelligence platform built for Indian users.

---

## 📌 Problem Statement

In India, millions of users struggle to understand business and financial news because:

- Content is too complex and technical  
- Mostly available in English  
- Same content shown to all users  
- No interactive explanation  

---

## 💡 Solution

MyET AI News Navigator transforms traditional news into:

✔ Simple  
✔ Personalized  
✔ Multilingual  
✔ Interactive  

---

## ⚙️ Key Features

- 🌐 Multilingual Support  
- 👤 Personalization (Student, Investor, Professional)  
- 🧠 AI Structured Briefing  
- 💬 Interactive Chat  
- 🇮🇳 India-Focused Insights  

---

## 🏗️ Architecture

Frontend: HTML, CSS, Bootstrap  
Backend: Node.js + Express  
Data: NewsAPI  
AI: Groq (LLaMA 3.3 70B)  

---

## ▶️ How to Run

1. npm install  
2. Add .env file  
3. node server.js  
4. Open index.html
